# Magnum - Enhanced Claude Code Desktop

A modern desktop application that provides an enhanced user interface for Claude Code with advanced features and intuitive design.

## What is Magnum?

Magnum transforms the Claude Code experience with a beautiful, native desktop interface that makes AI-powered development more productive and enjoyable. Built for developers who want the full power of Claude Code with a superior user experience.

## Installation

Download the appropriate installer for your platform from the Releases section:

- **Windows**: `.msi` installer
- **macOS**: `.dmg` disk image
- **Linux**: `.deb`, `.rpm`, or `.AppImage` packages

## System Requirements

- **Windows**: Windows 10 or later
- **macOS**: macOS 10.15 or later  
- **Linux**: Modern distribution with glibc 2.17+
- **Memory**: 512MB RAM minimum, 1GB recommended
- **Storage**: 100MB available space
- **Network**: Internet connection for Claude Code integration

## Prerequisites

Magnum requires Claude Code CLI to be installed and configured on your system. Visit the official Claude Code documentation for setup instructions.

## Getting Started

1. Install Claude Code CLI and configure your API access
2. Download and install Magnum for your platform
3. Launch Magnum and select your working directory
4. Start chatting with Claude through the enhanced interface
5. Explore the permission settings and usage analytics
